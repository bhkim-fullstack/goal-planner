# goal-planner

Goal Planner는 단순히 목표를 기록하는 앱이 아닙니다.

사용자가 목표를 이루는 과정에서 현재 어느 단계에 있는지 확인하고,
중간 목표와 기록을 통해 방향성과 성장 과정을 잃지 않도록 돕는 목표 관리 애플리케이션입니다.

목표 달성보다 중요한 것은,
왜 시작했는지 잊지 않고 꾸준히 성장하는 과정이라고 생각합니다.

<hr>

## 프로젝트를 만든 이유

사람들은 많은 목표를 세우지만,
현재 자신이 어느 단계에 있는지 체감하지 못해 중간에 포기하는 경우가 많다고 생각했습니다.

또한 목표를 이루는 과정에서
왜 시작했는지,
어떤 방향으로 성장하고 싶은지 잊어버리기도 합니다.

Goal Planner는 목표 달성 자체보다
성장 과정과 방향성을 기록하는 것에 집중한 프로젝트입니다.

<hr>

## 핵심 기능 

### 1. 진행도 시각화
사용자가 목표 달성까지 어느 정도 왔는지 직관적으로 확인할 수 있도록
Progress UI를 구현했습니다.

### 2. 기록 저장 기능
사용자가 중간에 멈추더라도
이전 진행 상태를 이어서 확인할 수 있도록 설계했습니다.


<hr>

## 주요 기능
- 목표 진행도를 0 ~ 100으로 시각화
- 단계별 중간 목표(Milestone) 관리
- 목표 진행 기록 저장
- 마지막 진행 상태 이어하기
- 성장 과정 히스토리 확인
